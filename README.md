## Sample Input & Output

Input:

G-enzyme framgments: ```AUCG,AUG,G,CU,ACUAUACG```

U.C-enzyme fragments: ```GGAC,U,AU,GAU,C,U,AC,GC,AU```

Output:

```
AUCGGACUAUACGAUGCU
AUCGAUGGACUAUACGCU
AUGGACUAUACGAUCGCU
AUGAUCGGACUAUACGCU
```
